A curated collection of resources to help you learn Nix and NixOS, from beginner to advanced.

## 🚀 Getting Started

### Official Resources
- [Nix Official Website](https://nixos.org/) — The official home of Nix and NixOS
- [Nix Manual](https://nixos.org/manual/nix/stable/) — Official Nix package manager documentation
- [NixOS Manual](https://nixos.org/manual/nixos/stable/) — Official NixOS documentation
- [Nixpkgs Manual](https://nixos.org/manual/nixpkgs/stable/) — Guide to the Nix packages collection

### Tutorials & Guides
- [Zero to Nix](https://zero-to-nix.com/) — A beginner-friendly introduction to Nix
- [nix.dev](https://nix.dev/) — Official tutorials and best practices
- [NixOS & Flakes Book](https://nixos-and-flakes.thiscute.world/) — Comprehensive guide to modern Nix

## 📚 Learning Nix Language

- [Nix Language Basics](https://nix.dev/tutorials/nix-language) — Official Nix language tutorial
- [Nix Pills](https://nixos.org/guides/nix-pills/) — In-depth series on Nix fundamentals
- [Nix By Example](https://nixbyexample.com/) — Learn Nix through practical examples

## 🏠 Home Manager

- [Home Manager Manual](https://nix-community.github.io/home-manager/) — Manage user environments with Nix
- [Home Manager Options Search](https://mipmip.github.io/home-manager-option-search/) — Search all Home Manager options

## ❄️ Nix Flakes

- [Flakes Introduction](https://nixos.wiki/wiki/Flakes) — NixOS Wiki guide to Flakes
- [Practical Nix Flakes](https://serokell.io/blog/practical-nix-flakes) — Real-world Flakes guide by Serokell

## 🔍 Search & Discovery

- [search.nixos.org](https://search.nixos.org/) — Search packages and options
- [NixOS Options Search](https://search.nixos.org/options) — Find NixOS configuration options
- [Home Manager Options](https://mipmip.github.io/home-manager-option-search/) — Search Home Manager options

## 🎥 Videos & Talks

- [NixCon Talks](https://www.youtube.com/@NixCon) — Conference talks from NixCon
- [The Nix Hour](https://www.youtube.com/playlist?list=PLyzwHTVJlRc8yjlx4VR4LU5A5O44og9in) — Weekly Nix Q&A streams

## 💬 Community

- [NixOS Discourse](https://discourse.nixos.org/) — Official discussion forum
- [NixOS Matrix](https://matrix.to/#/#community:nixos.org) — Real-time chat
- [r/NixOS](https://www.reddit.com/r/NixOS/) — Reddit community

## 📖 Books

- [NixOS & Flakes Book](https://nixos-and-flakes.thiscute.world/) — Free online book
- [Nix Pills](https://nixos.org/guides/nix-pills/) — The classic deep-dive into Nix

## 🛠️ Tools

- [devenv](https://devenv.sh/) — Fast, declarative development environments
- [direnv](https://direnv.net/) + [nix-direnv](https://github.com/nix-community/nix-direnv) — Automatic environment loading
- [nixpkgs-fmt](https://github.com/NixOS/nixfmt) — Nix code formatter
- [nix-tree](https://github.com/utdemir/nix-tree) — Visualize Nix derivation dependencies
